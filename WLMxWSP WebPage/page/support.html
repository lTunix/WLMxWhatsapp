Soporte

Al usar este cliente no se garantiza que a futuro no puedas ser vetado de su uso.
La política de Whatsapp no permite bots ni clientes no oficiales en su plataforma, por lo tanto esta app no debe considerarse del todo segura.

Para instalar el cliente obten la última versión en la sección de descarga.

Una vez completa la descarga, ejecuta el archivo .exe

Para usar el cliente, primero debes tener una cuenta de Whatsapp en tu teléfono.

Abre la aplicación y escanea el código QR que aparece en la pantalla de tu PC.

Una vez escaneado, podrás usar el cliente de Whatsapp en tu PC.

Para actualizar el cliente, descarga la última versión en la sección de "Descargar".

Una vez descargado, ejecuta el archivo .exe y sigue las instrucciones.

O también, puedes verificar si al abrir el cliente se encuentra una nueva versión disponible para instalar.

Si el cliente no carga correctamente tus contactos, intenta reiniciarlo.
A veces puede tardar debido a la alta cantidad de mensajes o chats.

Si el problema persiste, intenta cerrar sesión y volver a iniciarla.

Si el problema aún continúa, prueba a desintalar y volver a instalar.
