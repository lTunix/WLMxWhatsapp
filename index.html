WLMxWhatsapp

Un cliente de Whatsapp que busca revivir la estética, funcionalidad y experiencia de Windows Live Messenger.

Características

  • Interfaz de usuario de Windows Live Messenger.
  • Soporte para mensajes de texto, imágenes, emojis y stickers (Guiños).
  • Soporte para zumbidos (Limitado).
